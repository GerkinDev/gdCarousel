GDCarousel v1

GdCarousel est un script vous permettant de créer en une seule ligne un carousel coverflowé.

Aucune bibliothèque n'est nécessaire, oubliez jQuery!

Initialisation

var carousel = new gdCarousel(selector[, options]);
Nom Type Description
selector String chaîne de caractère ciblant l'élément racine du carousel
options Object 
{
    [time: {
        [main:  2500],  temps entre chaque rotation (en ms)
        [sub:   500],   durée de la rotation (en ms)
        [frame: 17]     temps entre chaque rafraîchissement (en ms)
    }],
    [displayed: 5],     nombre d'éléments affichés en même temps (doit être impaire)
    [autostart: true]   démarrage automatique lorsque la page est chargée
}

Fonctions

Pause

carousel.stop();

Démarrer

carousel.start();

Rotation immédiate

carousel.rotateNow();

Vos retours sont les bienvenus, pour tout bug ou demande!
